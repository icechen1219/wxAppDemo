# 威扬咨询小程序
### 开发配置
- 登录微信小程序管理界面
- 选择左侧“开发”菜单
- 右侧表单区选择“开发设置”
- 找到服务器域名栏
- 在request合法域名中添加如下域名：
```
https://api.map.baidu.com
https://douban.uieee.com
https://v1.hitokoto.cn
https://loveboyin.cn
```
- 在uploadFile合法域名中添加如下域名：
```
https://loveboyin.cn
```
- 在downloadFile合法域名中添加如下域名：
```
https://loveboyin.cn
```

### 完成上述配置后，即可在本地用微信小程序开发工具调试预览。
